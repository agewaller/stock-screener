対処法
  • クラウド > 0 / 端末 = 0: Firestore からの同期完了前にページを開いた可能性があります。ページをリロードしてください。
  • クラウド = 0 / 端末 > 0: 端末にデータはありますが、まだクラウドに保存されていません。ログイン状態で記録を追加するとクラウドにも保存されます。
  • クラウド > 0 / 端末 > 0: データは健在です。表示の問題であれば画面を再読み込みしてください。
  • クラウド = 0 / 端末 = 0: まだ記録がないか、全削除が実行されています。
  • エラーが出る: ネットワーク接続またはログイン状態を確認してください。
