| 戦略 | 採否 | 理由 |

## なぜ AdSense を採用しないか

### 1. YMYL コンテンツの審査リスク

Google AdSense の運用ポリシーは「Your Money or Your Life」（健康・金融・法律）コンテンツに極めて厳しい。

- 審査通過まで <strong>3-6 ヶ月</strong>
- 通過後も継続レビューで突然 demonetize される事例多数
- 医療アドバイス的な表現（「LDN は効く」等）はポリシー違反扱いになることがある

### 2. ユーザー体験の悪化

- ME/CFS 患者は感覚過敏が 60-70%（IOM 2015）。動的広告 / アニメーション / 音声広告は症状悪化の引き金
- スマホで広告タップ誤認 → 信頼喪失
- 競合サプリ・代替療法業者の広告が混入する可能性 → 患者を悪意ある業者へ誘導するリスク

### 3. SEO 評価への副作用

- Core Web Vitals: 広告で LCP / CLS スコアが悪化 → 検索順位ダウン
- Page Experience: 広告密度が高いと "intrusive interstitial" 判定でペナルティ

### 4. 運営者の倫理観

> 「闘っている人にとって辛くない」を判断基準にしている（about.html）

の方針と衝突。広告で運営費が出ても、患者からの信頼を失えば本末転倒。

## アフィリエイトの実装方針

### 配置箇所

各疾患 LP（10 ページ）の <code>&lt;footer&gt;</code> 直前に「📚 関連リソース」パネルを表示。
内容は <code>js/disease-affiliate-panel.js</code> の <code>CATALOGS</code> で疾患別キュレート（書籍・サプリ・デバイス）。

### 商品の選定基準（毎月レビュー）

1. **エビデンスがある**: 複数 RCT / メタ分析 / 専門家コンセンサス
2. **当事者開発者が実際に試したか、信頼できる患者会・専門医が推薦している**
3. **過剰な誇大広告がない**: 「治る」「完治」と謳う商品は除外
4. **過去 30 日 Amazon レビュー > 100 件、星 4.0 以上**
5. **国内法準拠**: 薬機法に抵触する薬剤・医療機器は除外

### 開示文（必須）

景品表示法（ステマ規制 2023.10.1）に準拠して、以下を全パネル冒頭に明示:

> ℹ️ <strong>本セクションには Amazon アソシエイト等のアフィリエイトリンクが含まれます</strong>。リンクから購入された場合、運営者に一定の手数料が支払われ、サービス運営費に充てられます（読者の支払額は変わりません）。記載商品の効果は個人差があります。購入前に主治医にご相談ください。

### リンク属性

```html
<a href="..." rel="sponsored noopener noreferrer" target="_blank">
```

- `rel="sponsored"`: Google の広告リンク識別（Schema.org / SEO 上推奨）
- `noopener noreferrer`: クリックジャック対策

### Amazon タグの設定

- 既定: `chroniccare-22`（CONFIG.AFFILIATE_NETWORKS）
- 上書き: `<body data-amazon-tag="your-tag-22">`
- 各 LP で個別の A/B テストも可能

## サプリブランド直接スポンサーシップ（将来）

### 候補ブランド

- iHerb（CoQ10 / B12 / セレン等の自社ブランド）
- DHC（国内サプリ）
- ファンケル
- 国内の機能性表示食品メーカー

### 条件（必須）

1. 疾患関連学会の専門医監修コメント付与
2. 「治療効果」を主張しない（薬機法対応）
3. 患者向けの透明な開示
4. ロイヤリティに関する明示（ロゴ表示・推薦文と購入リンクは別扱い）

### NG ブランド

- 民間療法系（高額の健康食品商法）
- 効能を断定する誇大広告系
- メールアドレス収集が主目的のもの
- 海外の規制対象になっている成分を含むもの

## 法人プラン（2026 年後半検討）

### ターゲット

1. **産業医契約の中規模企業**: 従業員 100-500 名、慢性疾患を抱える社員のセルフケア支援
2. **健保組合**: メンタルヘルス・健康経営施策の一環
3. **医療機関のクリニック**: 患者へのアプリ提案で受診間の症状追跡を改善

### プラン構成（草案）

- ¥3,000-5,000/ユーザー/月（産業医経由）
- 50 名一括契約から
- 産業医ダッシュボード（個別データは閲覧不可、傾向のみ可視化）

### 実装の前提条件

- 産業医 / 専門医インタビュー 5 件以上
- HIPAA / 3 省 2 ガイドライン正式準拠
- P マーク本取得
- 法人契約フォーム + 請求機能

## 寄付・後援（軽量導線）

ユーザーが「気持ちで支援したい」と言ってくれることがあるので、以下を整備:

- GitHub Sponsors（個人受け取り）
- Amazon ほしいものリスト（運営機材リクエスト用）
- 銀行振込（金額自由・寄付控除なしを明記）
- OpenCollective（透明性が必要な場合）
