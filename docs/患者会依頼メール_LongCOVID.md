```
〇〇 患者会 御中

突然のご連絡失礼いたします。
シェアーズ株式会社 代表の山口揚平と申します。

私は 2024 年に ME/CFS（筋痛性脳脊髄炎／慢性疲労症候群）を発症し、
2 年間の闘病経験を経て、Long COVID 患者と症状の重なりが多い ME/CFS
当事者の立場から、症状管理ツール『健康日記 (Health Diary)』を完全
無料で公開いたしました。

  https://cares.advisers.jp/long-covid.html

━━━━━━━━━━━━━━━━━━━━━━━
■ Long COVID 患者にとって特に意味のある特徴
━━━━━━━━━━━━━━━━━━━━━━━

1. Long COVID の多系統症状を漏れなく記録
   PEM（労作後倦怠感）、ブレインフォグ、起立性不耐症（POTS）、
   胸部症状、微熱、息切れなど、感染後の多彩な症状を横断的に
   追跡します。世界 6,500 万人と言われる Long COVID 患者の
   うち、約半数が ME/CFS 基準を満たすと報告されています。

2. AI による個別化アドバイス
   直近の記録に基づき、具体的なペーシング提案・生活改善・
   最新研究情報を日本語で提示します。Claude Sonnet / Opus
   を最新モデルに常時自動更新しています。

3. 主治医への提出用レポート自動生成
   90 日分の記録から、診察時に持参できるカルテ形式
   （主訴・症状経過・服薬・試した対処法・QOL）を Markdown で
   ワンクリック生成。「何を話せばいいか分からない」状態を
   解消できます。

4. ME/CFS・線維筋痛症・POTS・MCAS の併発にも対応
   Long COVID は併発疾患が多岐に渡ります。本アプリは複数疾患
   を同時選択して、それぞれに最適化された指標を同一画面で
   追跡できます。

5. 完全無料・広告なし・課金なし
   運営者が AI API コストを負担。データは Google Cloud 東京
   リージョンに暗号化保存、本人以外はアクセス不可です。
```
