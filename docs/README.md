## 文書の使い方

- **プライバシーポリシー / 3省2GL**: index.html の `showLegalPage('privacy')` にインラインで埋め込まれているほか、GitHub 上でも公開しています。両方を同期して更新してください。
- **世界患者数一覧**: `CONFIG.DISEASE_EPIDEMIOLOGY` のソース・オブ・トゥルース。UI/AI プロンプトから参照されます。
- **成長戦略 / SEO戦略**: 内部の戦略ドキュメント。運営者が参照する経営資料です。
- **マーケティング実行テンプレート**: コピー & ペーストでそのまま使えるよう完成原稿になっています。実際の送信作業は運営者が行います。

## 更新ルール

1. 文書を編集する際は必ず末尾の「改定履歴」に日付と変更内容を追加
2. 重要な改訂時は index.html の対応箇所も同時に更新
3. 年次レビュー (毎年 4 月) で全文書を見直す
4. 統計データ (世界患者数一覧) は WHO/GBD/IDF 等の更新に応じて随時改訂
