## 文書の使い方

- **プライバシーポリシー**: 本文の公開版は index.html 内の `showLegalPage('privacy')` にインラインで埋め込まれています。このマークダウンは編集用ソースです。両方を同期して更新してください。
- **3省2GL準拠宣言**: プライバシーと安全ページ（`/#privacy`）のロードマップカードから GitHub のこのファイルへ直接リンクしています。
- **Pマーク準備チェックリスト**: 公開用ではなく、内部の進捗管理用。審査準備に応じて随時チェックマーク (`❌` → `🟡` → `✅`) を更新してください。

## 更新ルール

1. 文書を編集する際は必ず末尾の「改定履歴」に日付と変更内容を追加
2. 重要な改訂時は index.html の対応箇所も同時に更新
3. 年次レビュー（毎年 4 月）で全文書を見直す
