## Instagram 投稿テンプレート

Instagram は画像主体なので、ストーリーズと投稿で使い分け:

### ストーリーズ (毎日の軸)

```
【画像: 軸の icon を大きく 🌿】
【テキスト】今日の新処方の軸: 食事・栄養
【スタンプ】リンクスタンプ → cares.advisers.jp
```

### 投稿 (週次)

```
【1 枚目: 週のハイライト】
「今週の ME/CFS 関連ニュース 3 本」

【2 枚目: 研究紹介】
【3 枚目: 治療選択肢】
【4 枚目: ユーザーの声 (許可制)】
【5 枚目: CTA】

キャプション例:
今週の健康日記まとめ。
ME/CFS 患者が毎日の記録で...(続き)

#ME_CFS #筋痛性脳脊髄炎 #慢性疲労症候群
#LongCovid #線維筋痛症 #POTS
#慢性疾患 #寛解への道 #健康日記
```

### 月次の ME/CFS 最新研究まとめ記事

```markdown
# 2026 年 X 月の ME/CFS 最新研究 5 選

この記事は、開発者の僕が先月読んだ ME/CFS・Long COVID 関連の論文や
研究の中から、特に患者にとって重要な 5 本をまとめたものです。

## 1. [論文タイトル]
- 著者: ...
- 掲載誌: ... (PMID: xxxxxx)
- 要約 (日本語): ...
- 患者への示唆: ...

## 2. [論文タイトル]
...

---

**今日から使えるアクション**
- 論文を踏まえて、今週から試せる小さな一歩は?
- 主治医に話してみる話題は?

---

※ この記事は [健康日記 (cares.advisers.jp)](https://cares.advisers.jp/) の
運営者が執筆しています。ME/CFS 患者向けの無料記録アプリも提供しています。
```
