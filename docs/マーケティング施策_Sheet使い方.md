# マーケティング施策一覧 — Google Sheets での使い方

**ソース**: `docs/マーケティング施策_全リスト.csv`（125 行・15 列・文字コード UTF-8）
**最終更新**: 2026-04-23

---

## 1. Google Sheets へインポート（30 秒）

### 方法 A — ブラウザから直接（推奨）

1. <https://sheets.new> を開く（新規スプレッドシート）
2. メニュー → **ファイル → インポート**
3. **アップロード** タブ → CSV をドラッグ
   - GitHub の生 URL: <https://raw.githubusercontent.com/agewaller/stock-screener/main/docs/マーケティング施策_全リスト.csv>
4. 「インポート場所」: **新しいシート**
5. 区切り文字: **カンマ**
6. テキストを数値や日付に変換: **オフ**（IDを文字列保持）

### 方法 B — IMPORTDATA 関数で常時最新

新しいシートの A1 セルに:

```
=IMPORTDATA("https://raw.githubusercontent.com/agewaller/stock-screener/main/docs/マーケティング施策_全リスト.csv")
```

リポジトリの CSV を更新すれば、Sheets 側も自動的に最新化されます（約 1 時間で反映）。

## 3. 推奨フィルタ／ビュー

### A. 「今日やるべきこと」ビュー

メニュー → データ → フィルタを作成 → 以下条件:
- 担当: **山口** OR **Claude→山口**
- 優先度: **最高** OR **高**
- 状態: **未着手**
- 期限: **Day 1** OR **Day 2** OR **Day 3**

→ 約 8-10 行に絞られます。これが「今すぐやる」リスト。

### B. カテゴリ別ピボットテーブル

挿入 → ピボットテーブル → 以下:
- 行: カテゴリ
- 値: ID の COUNT
- フィルタ: 状態 = 「未着手」

→ どのカテゴリに何件タスクが残っているかが一目で分かる。

### C. 月次予算ビュー

挿入 → ピボットテーブル → 以下:
- 行: 担当
- 値: 想定コスト(円) の SUM
- フィルタ: 状態 = 「未着手」

→ 各担当者が予算をいくら持ち出すべきか可視化。

### D. ガントチャート風（任意）

期限/頻度の列を Day → 日付に手動変換すれば、条件付き書式で簡易ガントが描けます。Sheets テンプレート「ガント チャート」とコピペで OK。

## 4. 状態の更新運用

| 状態 | 意味 |
|---|---|
| 未着手 | これから取り組む |
| 進行中 | 着手済み・完了未了 |
| 完了 | 一回完結のものは終了。継続施策（投稿等）は「進行中」のまま |
| 既存 | このプロジェクト前から存在 |
| 中止 | 検討の結果やらないことにした |

担当者は週次で自分の行を見直し、状態を更新するだけで OK。複雑な日付管理は不要。

---

## 5. 90 日後の振り返り

CSV をダウンロード → スナップショットとして保存 → 90 日後に同じ作業をして比較。

着実に「未着手」が減り、「完了」が増えていれば軌道に乗っています。逆に未着手が滞留している場合は、優先度の見直しや外注検討のサイン。

## 7. 追加施策の登録方法

新しい施策を思いついたら CSV の末尾に 1 行追加して `git commit` するだけ:

```csv
126,SNS/X,新しい施策名,未着手,山口,中,1h,0,期待効果,期限,関連ファイル,備考
```

リポジトリの CSV を更新する → IMPORTDATA 利用なら Sheets 側も自動反映。
