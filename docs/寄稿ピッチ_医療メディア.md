# 医療メディア寄稿ピッチ

**用途**: ヨミドクター / 日経 Health / Medical Note / m3.com 等への寄稿・取材依頼
**差出人**: info@bluemarl.in

---

## 📝 ピッチ戦略

医療メディアの編集者は毎日大量の売り込みメールを受け取っています。
**3 秒で読めるフック**で興味を引き、**本文 5 分以内で全容が伝わる**構成が必須です。

### 3 つの「刺さる切り口」

1. **当事者性** — 「開発者自身が ME/CFS 患者」という事実は他にない強み
2. **数字** — 「日本に 10-30 万人の ME/CFS 患者、しかし診断までに数年かかる」
3. **新奇性** — 「AI が毎日違う処方を出す」という技術的面白さ

## 📨 ピッチメール 1: ヨミドクター 宛

### 件名
```
【寄稿ご提案】ME/CFS 当事者エンジニアが開発した AI 症状記録アプリ — 日本 10-30 万人の患者に何をもたらすか
```

### 本文

```
ヨミドクター 編集部 ◯◯様

突然のご連絡失礼いたします。
シェアーズ株式会社 代表の山口揚平と申します。

ヨミドクターをいつも拝読しております。特に先日の
「[最近掲載された関連記事のタイトル]」の記事に感銘を
受けました。

私は 2 年前に筋痛性脳脊髄炎/慢性疲労症候群 (ME/CFS) を
発症した当事者であり、エンジニアとして 2 年間かけて
ME/CFS 患者向けの無料症状記録アプリを開発しました。

今回、以下の 2 点でヨミドクターの読者層にお役立ちできる
寄稿をさせていただけないかと、ご提案申し上げます。

━━━━━━━━━━━━━━━━━━━━━━━
提案 1: 寄稿記事「日本の ME/CFS 10-30 万人が
  直面する『診断までの数年』をどう短縮するか」
━━━━━━━━━━━━━━━━━━━━━━━

ME/CFS は WHO が 1969 年から ICD-11 で神経系疾患と
分類している正式な疾患ですが、日本では診断までに
平均 3-7 年かかるとされています。その理由は以下です:

1. 確定診断のバイオマーカーがまだない (IOM 基準は
   症状ベース)
2. 専門医が極めて少ない (全国で 50 名程度)
3. 患者自身が「何を記録すれば診断の助けになるか」
   を知らない
4. 医師と患者の対話時間が短く、複雑な症状を伝えきれない

特に 3 と 4 は技術で解決できる領域です。本アプリは
PEM (労作後倦怠感) の頻度・活動可能時間・HRV・ブレイン
フォグ等、ME/CFS 専門医が診断に使う指標を日次で記録し、
AI が主治医への質問テンプレートを自動生成します。

当事者として、また開発者として、この課題と解決のアプローチ
について、ヨミドクター読者 (50-70 代の健康意識の高い層)
にお役立ちする記事を 2,000-3,000 字で執筆可能です。

━━━━━━━━━━━━━━━━━━━━━━━
提案 2: 取材記事「ME/CFS 当事者エンジニアが
  無料 AI 記録アプリを開発した理由」
━━━━━━━━━━━━━━━━━━━━━━━

当事者インタビューとして、以下を取材いただけます:

- 発症のきっかけと初期の混乱
- 既存のアプリ・ノートでは足りなかったこと
- なぜ自分で作ろうと思ったか
- 2 年間のベータ開発期間で学んだこと
- 同じ症状の方へのメッセージ
- 無料公開・広告なしの事業選択の理由

取材は 60 分のオンライン面談 (Google Meet) で可能です。
体調により日程調整が必要な場合がありますが、最低 48 時間
の予告をいただければ確実に対応可能です。

━━━━━━━━━━━━━━━━━━━━━━━
資料一式 (プレスキット)
━━━━━━━━━━━━━━━━━━━━━━━

以下のリンクから詳細情報・スクリーンショット・開発者
プロフィール等をご確認いただけます:

・サービス: https://cares.advisers.jp/
・プレスキット: https://github.com/agewaller/stock-screener/blob/main/docs/プレスキット.md
・開発者 note: https://note.com/age
・開発者 X: https://x.com/yamaguchiyohei

━━━━━━━━━━━━━━━━━━━━━━━

本アプリは医療機器 (SaMD) ではなく、診断・治療の代替では
ありません。あくまで患者本人が日々の症状を記録・整理する
ツールです。全ての出力に「医師にご相談ください」の注意書き
を付し、YMYL 医療情報の扱いには細心の注意を払っております。

ヨミドクターの取材・寄稿にご関心をお持ちいただけましたら、
info@bluemarl.in までご返信いただけますと幸いです。
ご多用のところ恐れ入りますが、どうぞよろしくお願い申し上げます。

山口揚平
シェアーズ株式会社 代表取締役
info@bluemarl.in
https://cares.advisers.jp/
```

## 📨 ピッチメール 4: m3.com 宛 (医師専用 SNS)

### アプローチ
m3.com は一般のプレスリリースは受け付けず、MR 経由・広告経由でしか
医師にリーチできません。戦略:

1. m3.com の「MR 向け窓口」は使わない (広告費が発生)
2. 代わりに m3.com に寄稿歴のある医師 (ME/CFS 関連) を調べる
3. その医師に直接「患者に勧めていただけるツール」としてアプローチ
4. 医師が m3.com で「患者に勧めたツール」として言及してくれることを期待

具体的な連絡先候補 (敬称略):
- 山村隆 先生 (国立精神・神経医療研究センター、ME/CFS 研究班)
- 倉恒弘彦 先生 (関西福祉科学大学、慢性疲労研究)
- 松本美富士 先生 (線維筋痛症専門)
- 平澤裕 先生 (Long COVID 専門医)

## 🎯 記事掲載後のフォロー

1. **自社サイト・SNS で記事紹介** (記事への被リンクを送る)
2. **著者の方に感謝メール** (長期関係構築)
3. **記事掲載を社内・ファンに共有**
4. **Google Search Console でクリック数計測**
5. **次の寄稿依頼の打診** (3 ヶ月後)

---

## ⚠️ やってはいけないこと

1. ❌ 複数媒体に同じ内容で同時送付 (編集者は情報共有する)
2. ❌ 過度なフォローアップ (返信なしなら 2 週間待つ)
3. ❌ 取材中の医療断定発言 (「治る」「効く」等)
4. ❌ 競合アプリの批判
5. ❌ 特定の医師・クリニックへの誘導
6. ❌ 記事掲載後の SEO 要求 (リンク形式等への注文)
