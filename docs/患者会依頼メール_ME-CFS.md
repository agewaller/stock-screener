## 本文

```
〇〇 会 事務局 御中

突然のご連絡失礼いたします。
シェアーズ株式会社 代表の山口揚平と申します。私自身も 2 年前に ME/CFS
を発症し、今も症状と向き合いながら当事者として生活しております。

このたび、ME/CFS 患者と、同じように慢性疾患と闘う方々の日々の症状
記録を支えるウェブアプリ『健康日記 (Health Diary)』を、完全無料で
一般公開いたしました。

  https://cares.advisers.jp/me-cfs.html

━━━━━━━━━━━━━━━━━━━━━━━
■ アプリの特徴 — ME/CFS 患者にとって特に意味のある 5 点
━━━━━━━━━━━━━━━━━━━━━━━

1. ME/CFS 特有の指標に完全対応
   PEM 発生頻度（回/週）、1 日の活動可能時間、疲労度（1-10）、
   ブレインフォグ、HRV、NK 細胞活性、CRP、LDN の用量履歴まで、
   日本の ME/CFS 研究班で重視される項目を日次で記録できます。

2. AI による「今日の新しい打ち手」提示
   14 軸（食事・運動・瞑想・サプリ・睡眠・検査・先端医療・
   コミュニティ・専門医・バイオハック・東洋医学等）から、
   日付シードで毎日異なる処方を機械的に提示します。同じ助言の
   繰り返しを構造的に防ぎ、2 週間で全軸を体験できるしくみです。

3. 主治医提出用の「患者サマリーレポート」ワンクリック生成
   過去 90 日の記録から、診察時に持参できるカルテ形式
   （主訴・症状経過・服薬・試した対処法・QOL・主治医への
   質問事項）の Markdown を自動生成し、ダウンロード・印刷・
   コピーできます。

4. Long COVID、線維筋痛症、POTS、MCAS、EDS など併発疾患
   にも対応。複数疾患を同時に記録できます。

5. 完全無料 / 完全匿名可 / 広告・課金なし
   運営者が AI API コストを負担しています。データは Google
   Cloud 東京リージョンに暗号化保存され、本人以外は一切
   アクセスできません。

■ 透明性について
━━━━━━━━━━━━━━━━━━━━━━━

本アプリは医療機器（SaMD）ではなく、診断・治療・処方の代替では
ありません。あくまで患者本人が日々の症状を記録・整理するための
ツールです。すべての出力は参考情報として、医師と相談する材料と
してお使いいただくものです。

プライバシーについては、個人情報保護法と JIS Q 15001:2017 に
準拠した運用を行っており、プライバシーマーク（Pマーク）の取得
申請も準備中です。3 省 2 ガイドライン（厚労省・経産省・総務省の
医療情報取扱事業者ガイドライン）への準拠宣言書も公開しております:

  https://github.com/agewaller/stock-screener/blob/main/docs/3省2GL準拠宣言.md
```
