■ POTS 患者にとって特に意味のある点
━━━━━━━━━━━━━━━━━━━━━━━

1. 起立時心拍・血圧を日次記録
   臥位・立位それぞれの HR / BP を記録し、起立試験の基準
   （臥位 +30bpm 以上 or 心拍 120bpm 以上）の到達状況を
   可視化します。医師に渡す「起立性不耐症の客観指標」に
   直結します。

2. 水分・塩分摂取量の追跡
   POTS 対策の基本である塩分 8-12g／水分 2-3L／日の
   達成度を日次でチェック。コンプレッションストッキング
   使用の有無も記録可能。

3. 併発疾患への対応
   Long COVID 由来 POTS、ME/CFS 併発、EDS（血管型）との
   重なりにも対応。複数疾患を同時選択して記録できます。

4. 主治医提出用レポート ワンクリック生成
   循環器・自律神経の専門医を探す際に、今までの症状と試した
   対処法を整理したレポートを自動生成。

5. 完全無料・広告なし・課金なし
