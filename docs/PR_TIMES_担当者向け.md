## ⚠️ 注意事項・運営者向けチェックリスト

### 配信前の必須確認
- [ ] タイトル・本文に誤字脱字がないか (特に医療用語)
- [ ] 数値・統計の出典が正確か (WHO / GBD / 厚労省)
- [ ] 「治る」「効く」等の断定的医療表現がないか
- [ ] 会社情報 (設立年月・資本金・所在地) が正確か
- [ ] 連絡先 (info@bluemarl.in) が最新か
- [ ] OG 画像 (og-image.png) が準備済みか
- [ ] プライバシーポリシー・3省2GL宣言の URL が有効か
- [ ] 開発者 note / X リンクが有効か
- [ ] 配信日時が平日火-木の 10-11 時になっているか (記者が読む時間帯)

### 配信直後のアクション
- [ ] X で告知ツイート
- [ ] note で告知記事 (開発者アカウントから)
- [ ] 関係者 (家族、友人、支援者) に個別で連絡
- [ ] info@bluemarl.in を 1 日 3 回以上確認 (取材依頼対応)
- [ ] 最初の 24 時間は Google アラート通知を ON にしておく

### 配信後のフォローアップ
- [ ] 転載メディアをチェック (Yahoo ニュース、livedoor ニュース等)
- [ ] 取材依頼に 24 時間以内に返信
- [ ] Google Search Console で URL 検索ボリュームを確認
- [ ] 1 週間後に効果測定
