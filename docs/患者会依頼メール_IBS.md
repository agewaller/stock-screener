無料公開いたしました。IBS（過敏性腸症候群）の日常管理にも最適化して

━━━━━━━━━━━━━━━━━━━━━━━
■ IBS 患者にとって特に意味のある点
━━━━━━━━━━━━━━━━━━━━━━━

1. Bristol 便形状スケールでの日次記録
   排便の形状（1-7 スケール）、腹痛強度・部位、腹部膨満感、
   残便感を日次記録。IBS-D/IBS-C/IBS-M のサブタイプ判定に
   直結する指標をワンクリックで入力。

2. 食事と症状の相関分析
   食事記録（FODMAP タグ付き可）と腹部症状の時系列相関を
   AI が分析。個別のトリガー食材を見つける一助に。
   低 FODMAP 食の Phase 2（再導入）期の効果検証に最適。

3. ストレス・月経周期との連動
   IBS は腸脳軸・月経周期と強い関連があります。ストレス
   （1-10）・月経開始日を同時記録して相関を可視化。

4. 主治医提出用レポート ワンクリック生成
   消化器内科受診時に、過去 3 ヶ月の症状・試した食事療法・
   薬剤効果をまとめた Markdown レポートを自動生成。

5. 完全無料・広告なし・課金なし
