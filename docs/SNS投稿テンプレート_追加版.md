#### IBS-1: サブタイプ判定の重要性
```
🍽 IBS（過敏性腸症候群）でお悩みの方へ。

IBS-D（下痢型）/ IBS-C（便秘型）/ IBS-M（混合型）
で治療薬が変わります。

ラモセトロン → 下痢型
リナクロチド → 便秘型
ポリカルボフィル → 両方

Bristol スケールで 2-4 週間記録すると、医師との会話が変わります。

無料で記録 → cares.advisers.jp/ibs.html

#IBS #過敏性腸症候群 #健康日記
```

#### IBS-2: 低 FODMAP 食の三段階
```
低 FODMAP 食、Phase が分かれてるの知ってますか?

Phase 1: 除去 (2-6 週)
Phase 2: 段階的再導入
Phase 3: 個別化された永続食

Phase 1 だけで終えると栄養偏ります。
食事と症状を記録して再導入の効果を可視化しましょう。

#IBS #低FODMAP #過敏性腸症候群
cares.advisers.jp/ibs.html
```

#### IBS-3: 腸脳軸 + 月経連動
```
IBS の症状が「ストレスで悪化」「生理前に変動」と感じる方、
それは気のせいじゃないです。

📚 Rome IV 基準でも腸脳軸は中核機序
📚 月経周期と腸症状の連動は複数 RCT で実証

健康日記なら食事・ストレス・月経・症状を一画面で記録、AI が相関を分析。

cares.advisers.jp/ibs.html

#IBS #PMS #腸脳軸
```

### 橋本病（慢性甲状腺炎）— 300-500 万人がターゲット

#### 橋本-1: TSH 正常でも症状が残る方へ
```
🧬 橋本病で TSH は正常範囲なのに症状が残る方、
FT3 を測ってもらってますか?

T4 → T3 変換異常があると、TSH 正常でも
冷え・倦怠感・ブレインフォグが続くことがあります。

体温・体重・気分の日次記録 + 検査値推移で、
医師に「FT3 も見てほしい」と伝える材料に。

cares.advisers.jp/hashimoto.html

#橋本病 #甲状腺
```

#### 橋本-2: セレンと抗 TPO 抗体
```
複数の RCT で「セレン 200μg/日 で抗 TPO 抗体が低下」と報告。

ブラジルナッツ 2-3 粒で約 200μg。
ヨード過剰には注意（昆布出汁・海藻の大量摂取は避ける）。

抗体値の推移、健康日記で写真撮るだけで記録できます。

cares.advisers.jp/hashimoto.html

#橋本病 #甲状腺機能低下症
```

#### 橋本-3: 併発しやすい疾患
```
橋本病の患者さんへ。
自己免疫疾患は併発しやすい。

✓ 1 型糖尿病
✓ セリアック病
✓ 悪性貧血 (B12 吸収障害)
✓ ME/CFS・線維筋痛症

「甲状腺だけの病気」と思っていると見落とします。
健康日記なら複数疾患を同時記録できます。

cares.advisers.jp/hashimoto.html

#橋本病
```

## X — 開発者ストーリー（権威性 + 当事者性アピール）

### Story-1: なぜ無料か
```
「健康日記」のサービス、運営費は私個人が負担しています。

慢性疾患の患者さんは就労困難で経済的に苦しい方が多い。
そこに月額課金を提示するのは、自分が ME/CFS で苦しんでいた頃を
振り返ると残酷だなと感じました。

なので個人ユーザーは永続的に無料です。

cares.advisers.jp/about.html
```

### Story-2: 開発の動機
```
2024 年に ME/CFS（筋痛性脳脊髄炎）を発症してから 2 年。

「同じ病気と闘う人が、診察前夜に泣きながら症状を整理しなくても
よくなるアプリを作りたい」

その動機で書き始めたものが「健康日記」です。

開発者の話: cares.advisers.jp/about.html
```

### Story-3: 機能追加の順番
```
「健康日記」、機能を追加した順番:
```

## X — FAQ ページ流入用（検索クエリ「PEM とは」「LDN 処方」狙い）

### FAQ-1: PEM の正体
```
「動いた翌日に体調が崩れる」現象、それ "PEM" です。

📚 Post-Exertional Malaise（労作後倦怠感）
📚 ME/CFS の必須診断基準
📚 Long COVID 患者の約半数に出現

完全無料で記録できる: cares.advisers.jp/faq/pem.html

#ME_CFS #LongCOVID #PEM
```

### FAQ-2: LDN の処方医を探している方へ
```
LDN（低用量ナルトレキソン）を試してみたいけど処方医が見つからない、
というお問い合わせを受けることが増えました。

日本での LDN 処方の実情（保険外・自費・院内製剤）と、
始める前に記録しておくべき指標を 5 分でまとめました。

cares.advisers.jp/faq/ldn.html

#LDN #ME_CFS #線維筋痛症
```

### FAQ-3: ブレインフォグの伝え方
```
医師に「ブレインフォグ」と言っても伝わりにくい。

こう伝えるのがコツ:
❌ ぼんやりして集中できない
⭕ 集中力 30 分しか持たない、料理の手順が抜ける

具体的な日常困難 + 1-10 のスコアを 2-4 週分持参すると客観指標になります。

cares.advisers.jp/faq/brain-fog.html
```
