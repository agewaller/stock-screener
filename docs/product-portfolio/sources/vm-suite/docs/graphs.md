# ValuationMatrix グラフ一覧

## API エンドポイント

`global_code` は `{ticker}.{country_code}` 形式（例: `7203.JP`, `130A.JP`）。`.{country_code}` 省略時は JP 扱い（`7203` = `7203.JP`）。

### 財務系（listing 不要）

```
GET /companies/{global_code}/graphs/business_model     # ビジネスモデル分析
GET /companies/{global_code}/graphs/corporoid          # コーポロイド
GET /companies/{global_code}/graphs/prediction         # ロボット予測
GET /companies/{global_code}/graphs/value_chain        # バリューチェーン分析
GET /companies/{global_code}/graphs/bs                 # 貸借対照表
GET /companies/{global_code}/graphs/pl                 # 損益計算書
GET /companies/{global_code}/graphs/cf                 # キャッシュフロー計算書
GET /companies/{global_code}/graphs/roic               # 経営効率分析(ROICツリー)
```

### 株価依存（listing_id クエリパラメータ、省略時はデフォルト listing）

```
GET /companies/{global_code}/graphs/dcf                # ざっくり企業価値評価
GET /companies/{global_code}/graphs/multiples          # 倍率評価
```

### 共通クエリパラメータ

| パラメータ | 対象 | 例 |
|-----------|------|-----|
| `listing_id` | dcf, multiples | `?listing_id=123` |
| `years` | bs, pl, cf, roic, value_chain | `?years=20` (20/10/5/3、省略時は DB にある全年) |
| `quarter` | pl | `?quarter=true` |
| `compare` | 競合比較対応グラフ全て | `?compare=7267,7201` |
| `left` / `right` | pl | `?left=sales&right=business_profit` |
| `type` | value_chain | `?type=leverage` |
| `axis` | value_chain (compare時) | `?axis=absolute` or `?axis=ratio` |

---

http://valuationmatrix.com/companies/{code} で表示されるグラフ・ビジュアライゼーションの種類。

## 1. ビジネスモデル分析

企業の財務データと定性データをもとにシェアーズ株式会社の独自ロジックによって分析して、日本の上場企業を24個のビジネスモデルに分類したもの。あくまで財務諸表の特徴と概略から分類をしており、大まかな特徴として各モデルの持つ特徴を備えていると判断された結果としての分類。AI予測では、このビジネスモデルに応じた予測式を利用して企業の将来の売上・利益を推計している。

- 企業のビジネスモデルタイプを分類・表示（例: toC一般メーカー型）
- アイコン + テキスト説明

## 2. コーポロイド (Corporoid)

特性に応じ、企業を擬人化した画像。各企業ごとの詳細な特性の違いを武器・防具・若さ・体型・大きさ・背負うもの・レアリティで表現している。

- RPG ステータス風の企業パラメータ表示
- パラメータ: HP, ATK, DEF, SPD
- 年度別（FY2018, FY2019, FY2020 等）にキャラクター画像 + ステータス値を表示

## 3. ざっくり企業価値評価 (DCF)

買い物をするときに「これは高いor安い」をどのように判断していますか？おそらく、その商品の「価値」を考えますよね。企業の株を売り買いするときは「株主価値」を考えることが重要。「ざっくり企業価値評価（DCF）」ではすべての上場会社の「株主価値」がワンクリックで、すぐにわかる。

- DCF による割安度を星評価（1〜5）で表示
- 有料会員向け詳細あり

## 4. ロボット予測

企業の財務諸表とビジネスモデルから将来の売上・利益を計算した数字。あくまで過去の数字とモデルからの簡易的な予測であり、その正確性を保証するものではない。

- 業績予測の画像グラフ

## 5. バリューチェーン分析（単体）

バリューチェーン分析はビジネスプロセス分析とも言われることもあるとおり、「事業の各プロセスの状況」について確認できる機能。研究開発・仕入・生産・販売など各プロセスの財務数値上の状況変化が確認できる。本来、プロセスは会社ごと・事業ごとに異なるものを一般化し、かつ一般的な財務数値を使っているため、個社ごとの詳細分析の代わりにはなるほどの精度はない。概略や変化を掴む目的で利用する。ビジネスモデルによって・会社によっては表示されないプロセスもある。

カルーセル形式で各ステップごとに2軸チャート（左軸=棒グラフ:絶対値、右軸=折れ線:比率）を表示。

```
GET /companies/{global_code}/graphs/value_chain?type={type}
GET /companies/{global_code}/graphs/value_chain?type={type}&compare=7267,7201&axis=absolute
GET /companies/{global_code}/graphs/value_chain?type={type}&compare=7267,7201&axis=ratio
```

- compare なし → 単体2軸チャート（棒+折れ線）
- compare あり + axis=absolute → 複数社の絶対値を棒で比較
- compare あり + axis=ratio → 複数社の比率を折れ線で比較
- compare は最大5社

| type | ステップ | 棒（絶対値） | 折れ線（比率） | 折れ線単位 |
|------|----------|-------------|---------------|-----------|
| `leverage` | 資金調達 | 純資産 | 財務レバレッジ | 倍 |
| `rd` | 企画開発 | 研究開発費 | 研究開発費率 | % |
| `investment` | 投資 | 固定資産新規投資額 | 新規投資率 | % |
| `purchasing` | 材料調達 | 売上原価 | 原価率 | % |
| `manufacturing` | 生産 | 固定資産額 | 固定資産回転率 | % |
| `sales` | 販売 | 売上 | 一人当たり売上 | 百万円 |
| `operation` | 回収 | 営業CF | 売上債権回転月 | 月 |
| `dividend` | 還元 | 配当総額 | 配当率 | % |

## 7. 貸借対照表 (BS)

たとえば、家計の状態を見る時、「貯金」や「車」といった資産、「住宅ローン」などの負債を家計簿でチェックしますよね。貸借対照表（BS）は、企業の家計簿と同じ役割。企業の大きさや財産を知るのに便利な表。一般的に資産が大きく負債が小さい…つまり資本が大きい企業の家計は健全といえる。

- 積み上げ棒グラフ（資産・負債・純資産の構成）
- 期間切替: 20年分 / 10年分 / 5年分 / 3年分
- 競合比較対応

## 8. 損益計算書 (PL)

いくらものを売るのが上手でも、手元に利益が残らなくては意味がない。企業の場合も同じで、重要なのは売上高より「利益」。損益計算書（PL）は企業の儲け具合を知るのに便利なツール。営業利益は、本業によって得た利益のこと。売上高から原材料や人件費といった費用を差し引いて算出する。純利益は、営業利益から税金などを差し引いて最終的に会社に残る利益。個人で言えば額面ではなく「手取り収入」に相当する。営業利益が順調に伸びている会社がいい会社といえる。

- 2軸チャート: 左軸=棒グラフ、右軸=折れ線グラフ
- 期間切替: 20年分 / 10年分 / 5年分 / 四半期
- 競合比較対応

左軸・右軸はクエリパラメータ `left` / `right` で切り替え可能（デフォルト: left=sales, right=business_profit）:

| パラメータ値 | 項目名 |
|---|---|
| `sales` | 売上高合計 |
| `business_profit` | 営業利益 |
| `net_profit` | 当期純利益 |
| `sales_cost` | 売上原価合計 |
| `sga` | 販売費及び一般管理費 |
| `ebitda` | EBITDA |

## 9. キャッシュフロー計算書 (CF)

自分の手取り収入範囲内でお金を使っている人のほうが、賢い家計と言える。同じように企業の中には「やりくり上手」がいる。キャッシュフローとは一定期間の間に、企業に流入あるいは流出したお金の流れのこと。その中でも特に注目されるのがフリーキャッシュフローであり、これは稼ぎと投資の差。フリーキャッシュフローがプラスの領域にいる企業はやりくり上手と言える。企業価値を判断する際、こうしたやりくり上手な企業（＝フリーキャッシュフローが複数年継続してプラスな企業）がより高く評価される。

- 棒グラフ（営業CF, 投資CF, 財務CF 等）
- 期間切替: 20年分 / 10年分 / 5年分
- 競合比較対応

## 10. 経営効率分析 (ROIC ツリー)

税引後投下資本営業利益率（ROIC）は企業が投資した分に対して、どれだけリターンがあったかを示す指標。業種にもよるが、最低5％以上あったほうがよく、10％以上あれば優良企業といえる。慣れてきたらROICを構成するそれぞれの指標について、同業他社と比較してみると更なる発見があるかも知れない。

- ROIC を分解したツリー構造のグラフ
- 期間切替: 20年分 / 10年分 / 5年分
- 競合比較対応

## 11. 倍率評価

PERは株価収益率、PBRは株価純資産倍率のことを指している。これらは株価の割安度に関するもっとも一般的な指標。PERは10倍以下、PBRは1倍以下なら一般的に割安と言える。PER（企業が稼いだ純利益）、PBR（企業が投じた資本）に対して、それぞれの何倍の時価総額がつけられているかを表している。

- 営業利益倍率, PER, CF倍率, PBR の時系列グラフ
- 競合比較対応
