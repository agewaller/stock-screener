# 禅トラック（ZenTrack）｜ 外部向け 1枚資料

> **話すだけで、"意識の状態"まで見える化される。**
> ― 書く手間ゼロ。音声がそのまま、健康・仕事・信用・資産・時間のスコアと、その日の意識・注意の状態に。

## こんな方へ

- 日記・記録は続かない。でも **自分の1日を振り返りたい**
- 健康も仕事もお金も、**バラバラに気にして** 全体像が見えない
- 行動だけでなく **「どんな意識で過ごしたか」** まで知りたい
- Plaud などの音声デバイスを持っているが、**録ったきり** になっている

## 禅トラックでできること

- **録って送るだけ** … 文字起こしを専用アドレスにメールするだけで分析が走る
- **AI が5つの軸で日次スコア化** … 健康・仕事・信用・資産・時間
- **意識・注意の状態まで解析** … その日あなたの意識がどの次元に向いていたかを読み解き、総合スコアで可視化（他にない深さ）
- **タイムライン化** … その日の出来事を時間軸で自動整理
- **明日のアクション** … 蓄積データから、次にやることを提示

## 続けられる仕組み

- **入力が"話すだけ"** … 手で書く負担がないから続く
- **推移で見える** … 日々のスコアの変化を振り返れる
- **あなたのデータはあなたのもの** … 閲覧・削除・書き出しに対応

## はじめ方

1. アカウント登録（Google またはメール）
2. 専用メールアドレスが発行される
3. 音声デバイス（Plaud 等）の文字起こしを、そのアドレスに送るだけ
4. ダッシュボードに日次分析が表示される
