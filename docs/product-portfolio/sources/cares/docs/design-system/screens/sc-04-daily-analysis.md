## 設計メモ

- **「次の 1 アクション」を必ず 1 つ提示** (ADR-0013 S-7)
- 「もっと深く読む」= 深堀解析 (FN-AI-02)、1 日 1 回まで (BR-01)、本日実行済みならボタン非活性
- 月次コストキャップ到達時は 422 (BR-20) → モーダルで「今月の利用上限に達しました」
- 内容自体はサーバ側 fetch、ブラウザ state には極力残さない
- 「AI」リテラルは UI に出さない (BR-09)、「今日のひとこと」のような自然語に置換
