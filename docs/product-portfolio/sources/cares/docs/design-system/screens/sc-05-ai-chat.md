## 設計メモ

- **ストリーミング表示** (sequence-ai.md): SSE で chunk が届くたびにテキストが流れる
- メッセージは ai_messages テーブルに content BYTEA + AES-GCM 暗号化
- 添付 📎 = 画像 (画像解析 FN-AI-04 連携)
- メニューから:
  - アーカイブ / 復元 (状態遷移図参照)
  - モデル選択 (FN-AI-09)、ai_models テーブルから取得
  - 会話タイトル編集
- 月次コストキャップ到達 → 422 → モーダル
- 「AI」リテラル不使用、「相談」「ひとこと」等で代替 (BR-09)
- 会話履歴は SC-07 履歴 → 「相談」タブからも遡れる
