# SC-10 監査ログ閲覧

FN-AUDIT-01 (新規)。自分のデータに「誰がいつアクセスしたか」を本人に見せる。データ主権原則の中核。

## 関連

- FN-AUDIT-01 (監査ログ可視化)
- BR-13 (全アクセスを audit_logs に)、BR-18 (表示中メモリのみ)
- ADR-0007 (PII 禁止原則)

## 設計メモ

- `audit_logs WHERE target_user_id = me AND visible_to_user = TRUE`
- index `idx_audit_target_visible` で高速取得
- actor_type を日本語に表示変換:
  - `self` → 「あなた」
  - `admin` → 「管理者」
  - `system` → 「システム」
  - `integration` → 「外部連携 (サービス名)」
- 「管理者があなたのデータを read」のような行は本人にもしっかり見せる (BR-13, データ主権の核)
- IndexedDB にキャッシュしない (BR-18)、表示中メモリのみ
- `Cache-Control: no-store`
- スクロールで infinite load
- export 機能でこのログ自体もエクスポート可
