# SC-07 履歴 / 検索

過去の記録・解析・会話をカテゴリ別 / 日付別に振り返る。SC-02 ダッシュボードのタイムラインと異なり、検索・フィルタ・絞り込みが主役。

## 関連

- FN-DIARY-04 (タイムライン閲覧)、FN-DIARY-05 (個別編集・削除)
- FN-AUDIT-01 への導線あり

## ASCII ワイヤ

```
+--------------------------------------------------+
|  ← 戻る              履歴                         |
+--------------------------------------------------+
|                                                  |
|  [全て] [症状] [バイタル] [薬] [睡眠] [気分]      |
|        [食事] [検査] [活動] [写真] [メモ] [相談]   |
|                                                  |
|  期間: [2026/04] ～ [2026/05] ▼                  |
|                                                  |
|  キーワード検索: ┌──────────────┐ [🔍]         |
|                  └──────────────┘                |
|                                                  |
|  ──── 2026/05/27 (火) ────                       |
|  09:00  💊 ロキソニン 1錠          [編集] [削除]  |
|  08:00  🛏  睡眠 7.5h / 質 8         [編集] [削除]|
|                                                  |
|  ──── 2026/05/26 (月) ────                       |
|  21:00  📝 メモ「夕方から軽い頭痛」  [編集] [削除]|
|  18:30  🍽 夕食 (写真)              [編集] [削除] |
|  13:00  📊 血圧 122/78              [編集] [削除] |
|                                                  |
|  ... (中略)                                        |
|                                                  |
|  [もっと見る (古い順)]                            |
|                                                  |
|  ──────── ゴミ箱 (30 日以内) ────────             |
|  05/20 削除済み 「メモ」  [復元] [完全削除]        |
|                                                  |
+--------------------------------------------------+
|  🏠 ホーム  📅 履歴  ＋ 記録  💬 相談  ⚙ 設定     |
+--------------------------------------------------+
```

## 設計メモ

- カテゴリチップで絞り込み (複数選択可)
- 期間ピッカー (デフォルト直近 1 ヶ月)
- 「ゴミ箱」セクションで **ソフト削除中のレコードを表示**、復元 / 完全削除可能 (BR-21、状態遷移図参照)
- 編集タップで SC-03 (値プリセット)
- text_entries 等の暗号化カラムは表示直前にサーバで復号、ブラウザに残さない (ADR-0007)
- `Cache-Control: no-store` (BR-16)
- IndexedDB キャッシュ可 (暗号化、表示中のみ)
