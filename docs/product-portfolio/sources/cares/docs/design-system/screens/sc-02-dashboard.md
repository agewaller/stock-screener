# SC-02 ダッシュボード (タイムライン)

ログイン後のホーム。タイムライン (時系列の記録・解析・コメント統合表示) を中心に、主要アクションへの導線を持つ。

## 関連

- FN-DIARY-04 (タイムライン閲覧)
- FN-AI-01 (日々の解析、画面下部に最新結果)
- FN-DIARY-02 (記録入力への導線)

## 設計メモ

- タイムライン: 9 カテゴリ + テキスト + AI コメントを時系列に **1 行 1 行**
- 「今日のひとこと」= FN-AI-01、cognition-shifting output (具体的な次の 1 アクション提示、ADR-0013 S-7)
- 下部ナビゲーション = モバイルファースト
- フィルタは現在は最小、フェーズ 5 で「カテゴリで絞り込む」追加
- 個別タップで SC-03 編集モードへ / SC-07 履歴詳細へ
- Server Component で初期データを HTML に直接埋め込み、ハイドレーション後の React state には PII を残さない (ADR-0007 / project-cares-frontend.md)
- `Cache-Control: no-store` (BR-16)
