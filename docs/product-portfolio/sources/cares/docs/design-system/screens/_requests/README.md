# 画面 UI 修正依頼の置き場

画面ワイヤ (`../sc-XX-*.md`) の修正・追加要望を、議論の経緯を残す形でここに積む。

## ファイル命名

- 既存画面への修正: `sc-XX.md` (画面 ID と同じ)
- 新規画面候補: `sc-NEW-{slug}.md`

## 推奨テンプレ

```markdown
# SC-XX 修正リクエスト

日付: 2026-NN-NN
要望者: (名前 or 役割)

## 何を変えたい

(現在のワイヤと比較して、変えたい点を具体的に)

## なぜ

(理由・背景。ユーザーのフィードバック、運用での気付き等)

## 提案するワイヤ

```
+----+
| ... |
+----+
```

## 影響範囲

- 影響する FN: FN-XXX-NN
- 影響する API: GET /api/...
- 影響するテーブル: tables.X
```

## 反映フロー

1. このディレクトリに上記テンプレで md を作成
2. Claude / レビュアーが内容を確認
3. 合意後、対応する `../sc-XX-*.md` を更新
4. 不要になった依頼ファイルは削除 (もしくは「対応済み」マーカーを付けて残す)
