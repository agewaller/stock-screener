# SC-11 設定 (プロフィール / 通知 / アカウント削除)

ユーザー個人の設定。プロフィール / 言語 / モデル選択 / 通知 / アカウント削除 / 匿名昇格などを 1 画面にまとめる (タブ or アコーディオン)。

## 関連

- FN-PROF-01 (プロフィール編集)、FN-AUTH-05 (2FA)、FN-AUTH-07 (アカウント削除)、FN-AUTH-08 (匿名昇格)
- FN-INT-05 (エクスポートへの導線)、FN-AUDIT-01 (監査ログへの導線)
- アクティビティ: [`../../requirements/activity/activity-account-deletion.md`](../../requirements/activity/activity-account-deletion.md)

## 設計メモ

- アコーディオン: タップで展開、デフォルト「プロフィール」のみ開く
- **アカウント削除は最下部に隔離**、警告と猶予説明を併設 (BR-21)
- 「削除を申請する」→ インライン確認モーダル (BR-08、`confirm()` 不使用) → sequence-deletion-request.md
- 匿名から正規昇格 (FN-AUTH-08): 匿名ユーザーのみ表示、Firebase `linkWithCredential` で Google / email-password アカウントに昇格（UID 不変、既存データ継承）。**設計のみ・未実装**
- `current_medications` / `allergies` は BYTEA 暗号化、画面では復号して表示・編集
- 2FA 設定 (FN-AUTH-05): Identity Platform の MFA (SMS / TOTP) 登録フロー。**設計のみ・未実装**
- 連携サービス (FN-INT-04): Google Calendar の OAuth 再認可・解除 UI
