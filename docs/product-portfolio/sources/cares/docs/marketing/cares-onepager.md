# 健康日記（Cares）｜ 外部向け 1枚資料

> **記録するほど、あなたの体調が"伝わる言葉"になる。**
> ― 症状・薬・受診のメモが、医師に・家族に・自分のために役立つ要約へ。

## こんな方へ

- 通院のたびに「**前回どうだったか**」をうまく説明できない
- 体調・薬・食事の記録が **バラバラ** で振り返れない
- 家族に状態を伝えたいけれど、**専門的すぎて伝わらない**

## 健康日記でできること

- **かんたん記録** … 体調・症状・薬・食事・睡眠などを、その日のまま残せる
- **AI がやさしく整理** … 溜まった記録から、変化や気づきをまとめて提示
- **3つの"伝わる要約"を自動作成**
  - 🩺 **医師向け** … 経過と質問事項を、事実ベースで整理
  - 👪 **家族向け** … 専門用語を避け、いまの状態と必要な支えを平易に
  - 💬 **共有用** … 個人が特定されない、前向きなトーンで

## 安心して使えます（あなたのデータはあなたのもの）

- **データ主権** … 記録はいつでも閲覧・編集・1件ずつ削除・書き出しが可能
- **プライバシー設計** … 個人情報をブラウザに残さない設計。暗号化して保管
- **囲い込みません** … いつでも全データをエクスポートできます

## 料金・はじめ方

- **まずは無料でお試し**（登録前に「お試し AI 分析」を体験できます）
- 続けたい方向けの **月額プラン** をご用意（価格は ＿＿＿＿）
- スマホ・PC のブラウザですぐに開始：**cares.advisers.jp**

## ご注意

本サービスは体調の **記録・整理を助けるツール** です。診断・治療を行うものではありません。気になる症状は医療機関にご相談ください。
