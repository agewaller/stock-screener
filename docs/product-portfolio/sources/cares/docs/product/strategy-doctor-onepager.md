# cares — 医師向け 1 枚（外来効率化と患者の自己管理支援）

> 慢性疾患の患者さんの「受診前の経過整理」と「自己管理の継続」を支えるアプリです。診断・治療を行う医療機器ではありません。

## これは何か
患者さんがスマホで体調・服薬・睡眠・食事などを記録し、**受診前に経過を整理した「医師提出レポート（PDF）」**として持参・共有できる健康日記です。AI が記録を読み、患者さんの言葉で振り返りを返します。

## 先生のメリット（外来効率化）
- **経過が整理された状態で来院**：症状の推移・服薬状況・困りごとが PDF 1 枚にまとまる → 問診の時間を本質に使える。
- **自己管理の継続を後押し**：日々の記録が習慣化し、外来間の状態が見える。
- **書類負担の軽減**：補助金・各種支援申請の経過まとめ（下書き）も生成 → 先生への相談がスムーズに。

## 患者さんのメリット
- 「義務的な記録」ではなく、生活の点描として無理なく書ける（尊厳に配慮した設計）。
- 自分のデータを書き出し・復元できる安心（**広告や二次利用はしません／暗号化保存**）。
- つらいときの**緊急相談窓口・社会資源**への導線。

## 安全への配慮
- AI は**診断・処方をしません**。「相談すると良いかもしれない」程度の促しに留めます。
- **危険な助言を防ぐ安全制約**を設計（例：ME/CFS の方に過度な運動を勧めない＝PEM 配慮、服薬の自己判断変更を促さない、レッドフラグ症状では受診導線を優先）。
- 生成物（レポート・申請下書き）は**下書き**で、最終確認を促します。

## 医師提出レポートの中身（例）
- 総括（AI による経過の要約）
- 症状・バイタル・服薬・睡眠・食事・気分の明細（日本語ラベル・臨床的な並び）
- 対象期間（直近 7/30/90 日 等）

## ご相談したいこと
- 先生の外来で「あると役立つレポート様式」のフィードバック（項目・並び・粒度）。
- 患者さんへのご紹介（推奨）可否。**先生の推奨が患者さんの安心に直結**します。
