# トーン & マナー ガイドライン（#35 尊厳に配慮したライフログ志向）

実機テスター（Frances H さん）フィードバック起点。「病状を記録してください」と促すだけだと
**義務的な作文**に感じられ、病状しか残らない。自尊心の強い当事者は「病人扱い」に強く抵抗する。
→ 病状のデータ化はベースに保ちつつ、**日々の生活の点描（ライフログ）も自然に残せる**体験にする。

## 原則

1. **義務化しない**：「記録してください」「入力してください」と命じない。「よかったら」「ひとことから」
   のように、書かないことも選べる前提の語りかけにする。
2. **病人扱いしない**：主語を「症状」に固定しない。体調は数ある記録対象の **1つ**に過ぎない扱いに。
   食事・気分・出来事・うれしかったことなど、生活の側面を**先に**、対等に並べる。
3. **ライフログへ誘導**：問いかけ・プレースホルダは「今日はどんな一日でしたか？」のように
   日常の語り口に寄せ、体調はその中の一要素として書ける形にする。
4. **AI 応答も寄り添い基調**：BR-09 に従い「AI」「分析」等の技術ワードは出さない。
   評価・指導ではなく、共感とねぎらいを基調に。「次の1アクション」も押し付けず提案に留める
   （ADR-0013 S-7）。
5. **強制・ノルマ感を避ける**：連続記録などの仕掛けは「励まし」であって「ノルマ」ではない文言にする
   （責めない・煽らない）。

## 文言の指針（Do / Don't）

| 場面 | ❌ Don't | ✅ Do |
|---|---|---|
| 入力欄 | 「病状を記録してください」 | 「今日はどんな一日でしたか？ なんでも大丈夫」 |
| 空状態 | 「最初の体調を記録しましょう」 | 「今日のことを、ひとことから」 |
| 促し | 「毎日記録しましょう」 | 「書ける日に、書けるぶんだけ」 |
| 評価 | 「記録が不足しています」 | （責めない。何も言わない or ねぎらう）|
