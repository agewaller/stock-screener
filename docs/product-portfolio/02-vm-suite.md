# バリューマトリクス（Value Matrix / vm-suite）― 製品詳細

- **一言**: 企業バリュエーション分析（バイサイド水準の投資レポート・DCF）を AI で自動生成するプラットフォーム。
- **リポジトリ**: `shares-dev/vm-suite`（モノレポ。外部AIからは参照不可のため本書に詳細を集約）
- **対象**: toB（会計事務所・士業、M&A/FAS/監査法人、投資顧問・金融、事業会社IR・経営企画）／ **収益**: toB 契約
- **状態**: 開発・PoC段階。**最優先プロダクト（先に売上を作る）**

## 1. ビジョン・顧客価値

アナリストが数日かける企業価値評価を、**一次情報ベースで・一定品質で・数十分**に圧縮する。

**顧客の痛み:**
- 株価算定・バリュエーションに毎回数日かかり属人化
- 有報・四半期・適時開示を手で集計し、根拠資料づくりに時間が溶ける
- DCF・類似会社比較の前提のばらつきを説明するのが大変

**提供価値:** 入力は企業コードのみ。EDINET/TDnet/jQuants の一次情報を自動収集・構造化し、財務分析・DCF・多面相対評価・戦略マップ・投資判断まで一括出力。

## 2. 詳細な機能

### 分析エンジン（社内 AI スキル群として実装）
- **vm-report**: バイサイド投資アナリスト水準の深掘りレポートを §0〜§11 で生成（実態把握→ビジネスモデル→PEST/7S→財務→ベースDCF→2軸戦略マップ(SVG)→戦略オプション別DCF→リスク→投資判断）。EDINET注記・TDnet開示・IR PDF(中計/統合報告書/決算説明)・Web一次情報を統合
- **vm-dcf**: DCF valuation を1ショット算出。WACC/g を推定し Bear/Base/Bull の3シナリオで EV・非事業資産・有利子負債・株主価値・希薄化後一株価値を JSON 出力。多層検証（数式再計算・レンジ・過去整合・時価総額クロス）でハルシネーション抑制
- **vm-handson**: 四半期5年（20Q）財務を必須に、微小兆候抽出・粉飾深掘り・経営者の誠実さ点検・DCF不使用の多面相対評価で 0〜⑪章フル分析
- **vm-standard-all**: 10論点フル分析を vm00〜vm10 の11ブロックで出力
- **vm-analyze**: 企業コード＋モジュール番号で DB のプロンプト＋財務データを使い分析
- **peer-finder / related-tickers**: 比較peer銘柄の絞り込み・関連銘柄抽出

### データ基盤（DBテーブル＝マイグレーションから確認できる実装範囲）
- `statements`（EDINET財務正本、四半期含む）、`company_analyses`、`dcf_runs`（＋narratives）、`vm_reports`、`valuations`、`edinet_text_blocks`、`tdnet_disclosures`、`web_search_results`、`document_chunks`、`due_diligences`、`user_favorite_companies`、`invitations`、`settings` ほか
- データソース: **EDINET API v2**（有報・四半期）、**TDnet**（適時開示）、**jQuants**（株価）、IR PDF、Web一次情報

### アプリ構成（apps/）
| アプリ | 技術 | 役割 |
|---|---|---|
| `api` | Rust + Axum + SeaORM + PostgreSQL | REST API & バッチジョブ |
| `cli` | Rust | 分析パイプライン実行・取り込み（build-pack 等） |
| `user-console` | SolidStart + Prisma + Firebase Auth + PandaCSS + Ark UI | ユーザー向け閲覧（/dashboard, /companies, /companies/[code], /settings） |
| `admin-console` | SolidStart | 管理画面 |
| `vm-app` | SolidStart + Tauri | デスクトップアプリ |

### インフラ
Google Cloud Run / Cloud Build / Artifact Registry、Firebase Auth、PostgreSQL、DBマイグレーションは geni / Atlas(HCL)。

## 3. 差別化

- **一次情報ベース**（開示資料に根拠／憶測ではない）
- **多層検証**でハルシネーション抑制（数値の暴走を機械的にチェック）
- **速度と再現性**（人・タイミングに依存せず一定品質を量産）
- **そのまま使える成果物**（レポート・DCF JSON・戦略マップ SVG まで一括）

## 4. ターゲット顧客（売れる順の仮説）

1. **会計事務所・税理士法人**（非上場株価算定・事業承継＝定常業務、説明で売れる）
2. **M&Aアドバイザリー / FAS / 監査法人(バリュエーション)**（DD・PPA）
3. **投資顧問・IFA・地銀/信金の運用部門**（リサーチ工数削減）
4. **事業会社の経営企画・IR**（競合・自社価値分析）
5. **（裾野）個人のプロ投資家**（セルフサーブ入口）

## 5. マーケティング戦略（最注力）

1. **武器化**: `vm-report` で有名企業（例: トヨタ 7203 のサンプルが docs にある）のサンプルレポート3〜5本を生成→**無料DL（リード獲得）＋ブログ記事化（SEO）**。四半期ごとに注目銘柄を1本無料公開＝継続リードマグネット（エンジニア工数低＝自動生成）
2. **アウトバウンド（ファウンダー営業）**: ターゲットリスト→個別レター＋サンプル添付→「無償で1本作る」オファー→**有償PoC**へ
3. **ウェビナー**: 「AIで企業価値評価をどこまで自動化できるか」を月1（士業会・地域金融向け）。録画を資産化
4. **パートナー**: 会計ソフト/士業向けツール、M&A仲介プラットフォームと紹介提携
5. **セルフサーブ入口**: 個人プロ向け Pro プランを Web で即課金
