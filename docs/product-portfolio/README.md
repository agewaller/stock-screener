## プロダクトの価値（問い合わせ向けコピー・各200〜300字）

> 集客・営業・SNS でそのまま使える確定コピー。3つの価値の違い（見抜く／寄り添い導く／意識まで解析する）を明確に書き分けている。

**バリューマトリクス**
> 「この会社、本当に買っていいのか？」——その問いに、独自の視点で答えます。企業を**粉飾の可能性・経営者の誠実さ・事業構造（ROIC対WACC）・進化と退化・株価の触媒・価値創造余地**など多面の軸で採点し、DCF・戦略マップ・デューデリジェンスまで一気通貫で出力するAIアナリスト。単なる割安判定ではなく、“数字の裏”と“経営の質”まで踏み込んで見抜くのが他にない強みです。すべて一次情報に基づき、多層検証で精度を担保。今後はM&A・士業向け連携も拡大予定。まずは気になる1社で、無料サンプルレポートをお試しください。

**健康日記 Cares**
> 健康日記は、ただ記録するアプリではありません。綴った体調や気持ちの**蓄積を読み解き、「いまのあなたに最適で、最新で、ひと味違う気づき」**と、今日試せる具体的な一歩をそっと返す“寄り添い型”のパートナーです。同じ助言の繰り返しではなく、あなたの変化に合わせて提案そのものが進化します。さらに、医師に伝わる経過まとめや、家族向けのやさしい要約も自動作成。記録は暗号化して安全に保管し、いつでも閲覧・編集・書き出しが可能で、データはすべてあなたのもの。登録前の「お試し分析」から、無料で始められます。

**禅トラック ZenTrack**
> 禅トラックは、**話すだけ**であなたの“意識の状態”まで解析するライフログ分析です。音声メモを送れば、AIが一日を健康・仕事・信用・資産・時間の5軸でスコア化し、さらに**あなたの意識・注意がどの次元に向いていたかを読み解いて**、総合スコアと“明日の具体的な一手”を返します。表面的な行動だけでなく「どんな意識で過ごしたか」まで可視化されるのが、他にない深さ。書く手間がないから自然と続きます。Plaudなどの音声デバイスとは特に好相性。話すだけで自分を整える新しい習慣を、いまβ版で無料体験できます。

## 共通する設計思想（全プロダクト共通の DNA）

> **「ユーザーが対話しながらデータを溜め、その蓄積データを使って、より最適・最新・something new な結果を返し、
> ユーザーの認知を変えてアクションを促す」**

これは健康・お金・時間・仕事のどの領域のジャーナリング/AI/管理システムにも適用する共通方針。具体的には:

1. **すべて格納する**（正本はサーバ、localStorage はキャッシュ）
2. **いつでも個別に編集・削除できる**（全消ししかない状態を作らない）
3. **要約して使える（3つの受け手）**：医師/専門家向け・SNS向け・近親者向け
4. **認知を変える出力**：記録の反復ではなく、次の具体アクションを1つ提示
5. **エクスポート/ポータビリティ**（ベンダーロックインしない）

加えて **鍵・秘密情報はサーバ側のみ**（ブラウザ・DB に置かない）、**管理者を絶対にロックアウトしない**を厳守する。
