# バリューマトリクス（Value Matrix）｜ toB 営業 1枚資料

> **AIが、バイサイド水準の企業価値評価レポートを"数十分"で。**
> ― アナリストが数日かける分析を、一次情報ベースで一定品質に。

## こんな課題はありませんか？

- 株価算定・バリュエーションに **毎回数日** かかり、属人化している
- 有報・四半期・適時開示を **手で集計** し、根拠資料づくりに時間が溶ける
- DCF や類似会社比較の **前提のばらつき** を説明するのが大変

## バリューマトリクスができること

| 入力 | 処理 | 出力 |
|---|---|---|
| 企業コードを指定 | EDINET / TDnet / jQuants の一次情報を自動収集・構造化 | **深掘り投資レポート（§0〜§11）** |
| | AI が財務分析・DCF・多面相対評価を実行 | **DCF（Bear / Base / Bull 3シナリオ）** |
| | 多層検証でハルシネーションを抑制 | **2軸戦略マップ（図）／投資判断** |

## 選ばれる理由（差別化）

- **一次情報ベース** … 開示資料に根拠を置く。憶測ではない
- **多層検証** … 数式再計算・レンジ・過去整合・時価総額クロスで数値の暴走を抑制
- **速度と再現性** … 同じ品質を、人とタイミングに依存せず量産
- **そのまま使える成果物** … レポート・DCF・戦略マップまで一括出力
