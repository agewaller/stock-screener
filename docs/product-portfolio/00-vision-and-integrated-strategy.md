# 統合ビジョンと統合マーケティング戦略

## 1. ビジョン

**「対話しながらデータを溜め、AI が最新・最適・something new を返し、人の認知を変えて次の行動を促す」**
― この同じ思想を、領域別に実装したのが3プロダクトである。

- **vm-suite = 企業価値**のジャーナリング/AI分析（toB）
- **cares = 健康**のジャーナリング/AI分析（toC）
- **zentrack = 生活全体（健康・仕事・信用・資産・時間）**のジャーナリング/AI分析（toC）

3つはバラバラの製品ではなく、**1つのコーポレートブランド/ブログの下にぶら下げ**、SEO・ブランド・実績を共有資産にする。

## 3. 中核戦略：vm-suite 先行

少人数・低予算では「**単価が高く・説明で売れ・既に武器がある**」プロダクトから売上を作るのが定石。

| 観点 | vm-suite (toB) | cares / zentrack (toC) |
|---|---|---|
| 単価 | 高（年数十万〜数百万） | 低（月数百〜数千円） |
| 黒字化に要する顧客数 | 数社 | 数千〜数万人 |
| CAC回収 | 商談で説明でき、回収が速い | 広告依存、回収に時間 |
| 既存の武器 | **レポート自動生成がそのままデモになる** | プロダクト体験勝負 |
| 少人数との相性 | ◎ ファウンダー営業が効く | △ 広告投下力が要る |

→ cares / zentrack は当面「種まき」。広告は最小限で、コンテンツとオーガニックでユーザーと学びを溜める。

## 9. リスク・注意

- **toBの肝は信頼と精度**：AIレポートの数字のハルシネーションは致命傷。デモ前に多層検証を必ず通し「一次情報ベース」を訴求。
- **cares は規制感度が高い**：医療広告ガイドライン/薬機法に注意。「診断・治療」を謳わず「記録・整理ツール」として訴求。データ主権は強い差別化メッセージ。
- **計測なしで広告を焚かない**：P0 の計測基盤が立つまで有料広告は出さない。
- **予算は人件費別**：外注は予算内、専属採用は売上連動で別途判断。
