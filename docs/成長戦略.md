## 戦略の全体像 — 3 層構造

```
┌─────────────────────────────────────────┐
│ Layer A: プロダクト内蔵ループ (実装済み)    │
│ - シェアボタン (X/LINE/共有/コピー)        │
│ - ストリーク + 達成バッジ                  │
│ - 今日の軸プレビュー                       │
│ - 公開ユーザー数 (社会的証明)              │
│ - リファラル URL                            │
│ - 初回ユーザー向け 3 ステップ歓迎             │
├─────────────────────────────────────────┤
│ Layer B: コンテンツ / SEO (実装済み)        │
│ - ME/CFS 完全ガイドをトップ静的コンテンツに │
│ - JSON-LD MedicalCondition / Article       │
│ - プライバシーポリシー / 3省2GL 準拠宣言     │
│ - 世界患者数データ                          │
├─────────────────────────────────────────┤
│ Layer C: 外部施策 (運営者アクション必須)    │
│ - note 記事からの被リンク設置                │
│ - SNS 運用 (X / Instagram)                 │
│ - 患者会・医療メディアへの掲載依頼           │
│ - プレスリリース配信                        │
└─────────────────────────────────────────┘
```

## 実装済みの獲得ループ (Layer A)

### ① シェアボタン (X / LINE / 共有 / コピー)
- 場所: AI 分析結果カード内の「新しい打ち手」ブロック直下
- 仕組み: 今日の軸 + AI の new_approach を自動的に 140 字以内に整形
- プレテキスト例:
  > 【今日の新処方 / 🌿 食事・栄養の新戦略】
  > 朝食前に発酵食品 (味噌汁 200ml) を加えて腸活を始める。2 週間継続で便通の変化を観察。
  > 慢性疾患と寄り添う AI 記録アプリ「健康日記」 #ME_CFS #慢性疾患 #健康日記
  > https://cares.advisers.jp/?ref=abc12345
- ref 付き URL で招待トラッキング可能

### ② ストリーク + 達成バッジ
- 場所: ダッシュボード上部の大型ウィジェット
- 表示項目: 🔥 連続記録日数、📚 通算日数、📅 今日の指定軸
- バッジ: 🌱最初の一歩 / 🔥3日連続 / ⚡1週間連続 / 🌊2週間連続 / 🏔30日 / 🏆100日 / 📚📖🎓通算記録系
- 効果: 帰訪モチベーション、継続の可視化

### ③ 今日の軸プレビュー (ログイン画面)
- 場所: ログイン画面のヒーロー直下、ゲスト入力エリアの上
- 内容: 今日の 14 軸のうち 1 つ + 候補例
- 効果: 「今日は何の軸かな?」という帰訪の理由創出

### ④ 公開ユーザー数 (社会的証明)
- 場所: ログイン画面の今日の軸カード内右側
- ロジック: Firestore users collection の件数を lazy fetch (1 時間キャッシュ)
- 表示: 「🌱 X 人が使用中」

### ⑤ リファラル URL (`?ref=abc12345`)
- 仕組み:
  1. URL に `?ref=xxx` があると localStorage に保存
  2. ユーザーが新規登録した時点で `users/{uid}.referredBy` に書き込み
  3. 管理者が「ユーザ管理」タブで referredBy 集計を確認 (将来拡張)
- シェアボタン経由でユーザーの uid 先頭 8 文字が自動付与

### ⑥ 初回ユーザー向け 3 ステップ歓迎ウィジェット
- 表示条件: textEntries と symptoms が両方 0 件のとき
- 内容: 「① 1 行でも書く → ② 処方を受け取る → ③ 明日も続ける」
- 効果: First-time → Second-time 遷移率の向上

## 獲得チャネル別の戦略

### 🔍 オーガニック検索 (SEO)

**メインターゲット**: ME/CFS 関連クエリで検索している患者

**施策**:
- ME/CFS 完全ガイド (実装済み)
- 定期的なコンテンツ更新 (月 1 回)
- 関連疾患 (Long COVID, 線維筋痛症) のガイドも順次追加
- 詳細は `docs/SEO戦略_MECFS_1位獲得プラン.md` を参照

**想定 CAC**: 0 円 (コンテンツ制作のみ)
**想定効果**: 月 100-500 人の新規訪問

### 📱 SNS (X / Instagram / note)

**メインターゲット**: 慢性疾患アカウントをフォローしている患者

**施策**:
- X (Twitter) アカウント開設: `@kenkou_nikki_app`
- 毎日の「今日の新処方軸」をポスト (自動化案: Cloudflare Worker で Cron)
- 世界/日本の患者数データを視覚的なツイートに
- 開発者 (@age) の個人アカウントからも発信
- note 連載による SEO + 信頼性の同時獲得

**想定 CAC**: 0 円 (時間投資のみ)
**想定効果**: フォロワー 1,000 人で月 50-100 人の新規訪問

### 🤝 パートナーシップ

**メインターゲット**: 患者会・医療従事者・クリニック

**施策**:
- 筋痛性脳脊髄炎の会への掲載依頼
- 慢性疾患専門クリニック 10 院へのアプローチ
- プライマリケア医への紹介パンフレット (将来)
- 製薬会社のペイシェント・コミュニティとの連携 (将来)

**想定 CAC**: 紹介料 or 手数料
**想定効果**: 1 パートナーあたり月 5-20 人の新規訪問

### 📰 メディア PR

**メインターゲット**: 健康メディア読者

**施策**:
- プレスリリース配信 (PR Times、@Press)
- 「ME/CFS 当事者開発者が作った AI 記録アプリ」ナラティブ
- 健康メディア (ヨミドクター、日経 ヘルス、Medical Note 等) への寄稿
- 学会発表 (ポスター、デモ)

**想定 CAC**: 配信料 1-5 万円
**想定効果**: 1 メディア掲載で月 100-500 人

### 👥 コミュニティ

**メインターゲット**: 既存ユーザーとの対話

**施策**:
- Discord / LINE オープンチャットで患者コミュニティ
- 月 1 回のオンライン交流会 (Zoom)
- ユーザーフィードバックの可視化 (変更履歴ページ)
- 当事者ブロガー / インフルエンサーとの連携

**想定 CAC**: 0-1 万円/月
**想定効果**: 口コミで月 20-50 人
