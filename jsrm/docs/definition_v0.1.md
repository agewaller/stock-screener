# JSRM v0.1 定義書

## プロダクト名

- 日本語：日本参戦リスク・主権モニター
- 英語：Japan Sovereignty Risk Monitor
- 短縮名：JSRM

## 一言説明

公開情報に基づき、日本が戦争・準戦時体制に巻き込まれる構造的リスクを定点観測し、
リスク上昇時に希望者へ通知するダッシュボード。

## v0.1 のスコープ

- 8 カテゴリ × 重み付き集計
- Level 0〜5 のリスクレベル
- トレンド（前日比 / 7日 / 30日）
- 上昇要因トップ5
- メールアラート（ダブルオプトイン、解除リンク付き）
  - 4 種類の発火条件
  - 1 ユーザー 1 日 2 通までの上限
- GitHub Actions による毎時更新 + アラート送信

## v0.1 で**やらないこと**

- 個別ユーザーの詳細プロファイリング
- 機械学習による予測（指数は固定式の集計）
- 戦争確定の断定表現
- ニュース本文の長文転載
- 攻撃手法・軍事作戦・脆弱性利用の助言

## 倫理ガード

- リスク表示は「指数 / レベル / 変化方向 / 根拠 / ソース / 不確実性」の透明開示
- 通知は冷静な事実通知。恐怖を煽る赤文字や扇情的表現は使わない
- メールアドレスは公開 JSON に保存せず、Supabase に限定
- トークンは平文ではなく SHA-256 ハッシュで保存
