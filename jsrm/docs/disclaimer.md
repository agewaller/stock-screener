# 免責事項

本サイトおよびメール通知は、公開情報に基づくリスク観測であり、戦争発生を
予言・断定するものではありません。数値は政策判断・投資判断・避難判断の
唯一の根拠にしないでください。重要局面では政府発表、一次情報、専門家判断を
確認してください。

## 禁止事項

本サイトでは以下を提供・支持しません。

- 戦争発生を断定する表現
- 攻撃方法・軍事作戦・脆弱性利用に関する助言
- 特定国民・民族への憎悪扇動
- 未確認情報の断定的引用
- ニュース本文の長文転載
- メールアドレスの公開保存

## 個人情報

メールアドレスは公開 JSON には保存されません。Supabase の `subscribers`
テーブルにのみ保存され、Service Role Key を持つサーバー側からのみ操作されます。
解除はいつでも可能です。
